# 北邮人论坛收藏夹失效帖清理

一个 Tampermonkey 用户脚本，用于自动识别并批量删除北邮人论坛（BYR BBS）收藏夹中的失效帖子。

## 功能特性

- 自动识别收藏夹中标题为"[指定的文章不存在或链接错误]"的失效帖子
- **预扫描**：先遍历所有页面统计失效帖总数，再开始删除
- **进度条**：实时显示删除进度（百分比 + 预计剩余时间）
- **暂停/继续**：随时暂停清理，暂停时自动保存和导出记录
- **实时保存**：每删除一条立即写入 `localStorage`，防止数据丢失
- **导出记录**：随时点击"导出记录"按钮下载已删除帖子的 `.txt` 文件
- 逐条删除，自动处理确认弹窗
- 支持多页收藏夹，自动翻页继续清理
- 深色控制台风格 UI 面板，可最小化

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 `byr-cleanup-favorites.user.js` 文件，选择"Raw"查看原始内容
3. Tampermonkey 会自动弹出安装提示，点击"安装"即可

或者手动安装：打开 Tampermonkey 控制面板 → 新建脚本 → 粘贴 `byr-cleanup-favorites.user.js` 的全部内容 → 保存。

## 使用方法

1. 打开北邮人论坛收藏夹页面（`bbs.byr.cn/fav`）
2. 页面右下角会出现深色的 **"BYR 收藏清理"** 控制台面板
3. 点击 **"开始清理"** 按钮，脚本将先扫描所有页面统计失效帖数量
4. 扫描完成后自动开始逐条删除，进度条实时更新
5. 随时点击 **"暂停"** 暂停清理（暂停时会自动保存记录并下载备份）
6. 点击 **"继续"** 恢复清理
7. 点击 **"导出记录"** 随时下载已删除帖子的记录文件
8. 面板可通过标题栏按钮最小化为小图标，点击图标重新展开

## 注意事项

- 脚本默认匹配 `bbs.byr.cn` 域名。如果论坛域名有变动，请在 Tampermonkey 中修改脚本头部的 `@match` 字段
- 清理过程中请勿关闭或刷新页面
- 每次删除间隔约 300ms + 页面等待时间，避免请求过于频繁
- 删除记录会实时保存到浏览器 `localStorage`（key: `byr_cleanup_records`），即使页面意外关闭也不会丢失
- 暂停和完成时都会自动触发一次 `.txt` 文件下载
