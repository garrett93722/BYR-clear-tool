# 北邮人论坛收藏夹失效帖清理

一个 Tampermonkey 用户脚本，用于自动识别并批量删除北邮人论坛（BYR BBS）收藏夹中的失效帖子。

## 功能特性

- 自动识别收藏夹中标题为"[指定的文章不存在或链接错误]"的失效帖子
- 逐条删除，自动处理确认弹窗
- 支持多页收藏夹，自动翻页继续清理
- 删除完成后自动导出清理记录（含原帖标题和发帖时间）为 `.txt` 文件
- 页面右下角悬浮按钮，手动点击触发，不会误操作

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 `byr-cleanup-favorites.user.js` 文件，选择"Raw"查看原始内容
3. Tampermonkey 会自动弹出安装提示，点击"安装"即可

或者手动安装：打开 Tampermonkey 控制面板 → 新建脚本 → 粘贴 `byr-cleanup-favorites.user.js` 的全部内容 → 保存。

## 使用方法

1. 打开北邮人论坛收藏夹页面（`bbs.byr.cn/fav`）
2. 页面右下角会出现红色的 **"清理失效收藏"** 按钮
3. 点击按钮，脚本将自动开始清理
4. 清理完成后会弹出统计信息，并自动下载一份删除记录文件

## 注意事项

- 脚本默认匹配 `bbs.byr.cn` 域名。如果论坛域名有变动，请在 Tampermonkey 中修改脚本头部的 `@match` 字段
- 清理过程中请勿关闭或刷新页面
- 每次删除间隔约 300ms + 页面等待时间，避免请求过于频繁
- 建议在收藏数量较多时使用，少量失效帖手动删除即可
